**page**
--------

`0` - заставка

`1` - меню

`2` - описание

`3` - анимация

`4` - погрешности

`5` - таблица

`6` - график

`7` - помощь

`8` - подтверждение закрытия

`9` - сохранения

`10` - создать сохранение

---------

**memory**
----------

`newfilename` - имя файла

`error` - погрешность

`table` - таблица

---------

**log**
-------

`way` - указать путь лога в виде списка, не указывая log

`args` - указывать аргументы через запятую
